# decodeUTF8

> utf8 转字符串

---

#### 参数

| 参数名     | 参数描述             | 参数类型 | 是否必填 | 默认值 |
| ---------- | -------------------- | :------: | :------: | :----: |
| **string** | 要解码的 utf8 字符串 |  string  |    √     |        |

#### 返回值

| 输入类型 |   返回值   |
| :------: | :--------: |
| any case | **string** |

---

#### 代码示例

> 一般使用：

```js
import { decodeUTF8 } from 'mytils';

decodeUTF8('&#x5B9D;&#x8D1D;&#xFF0C;&#x56DE;&#x5BB6;&#x5403;&#x996D;&#x5417;&#xFF1F;'); // 宝贝，回家吃饭吗？
```
