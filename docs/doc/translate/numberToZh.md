# numberToZh

> 数字转中文大小写

---

#### 参数

| 参数名        | 参数描述     | 参数类型 | 是否必填 | 默认值 |
| ------------- | ------------ | :------: | :------: | :----: |
| **number**    | 要翻译的数字 |  number  |    √     |        |
| **isCapital** | 是否中文大写 | boolean  |          |        |

#### 返回值

| 输入类型 |   返回值   |
| :------: | :--------: |
| any case | **string** |

---

#### 代码示例

> 一般使用：

```js
import { numberToZh } from 'mytils';

numberToZh(12345678); // 一千二百三十四万五千六百七十八
numberToZh(1234506, true); // 壹佰贰拾叁万肆仟伍佰零陆
numberToZh(123.321, true); // 一百二十三点三二一
```
