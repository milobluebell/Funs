# 开始

> **Funs** 是前端开发中，常用到的功能型方法的集合。

比如 **时间格式的转换、省略小数点后 N 位、倒计时展示、笛卡尔乘积** 以及更多...

### 安装

#### npm

```shell
npm install funs --save
```

#### yarn

```shell
yarn add funs --save
```
